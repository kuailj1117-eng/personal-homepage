# 个人主页

全栈开发者的个人主页，纯 HTML/CSS/JS 构建，零依赖。

## 特性

- ✨ 粒子动画背景（Canvas）
- ⌨️ 打字机效果
- 🌓 暗色/亮色双主题切换
- 📱 响应式设计
- 🫧 毛玻璃卡片 UI
- 📜 滚动渐入动画
- 🔢 数字滚动统计
- 🎯 项目筛选

## 本地预览

```bash
cd personal-homepage
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 部署

部署到 GitHub Pages，push 即自动更新。

## 技术栈

HTML5 · CSS3 · 原生 JavaScript · 零框架
